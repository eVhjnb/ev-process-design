# Level 0 – ServiceCo Value Chain

This folder contains a Level 0 value-chain view for **ServiceCo** 
(an anonymized remote service / operations company).

The value chain typically includes:

- Demand Generation & Marketing
- Sales & Client Acquisition
- Recruitment & Talent Pool Management
- Matching & Placement
- Service Delivery & Operations
- Billing & Collections
- Finance & Controlling
- HR, People & Internal Operations
- Data & Analytics / Continuous Improvement

File:

- `serviceco_value_chain.puml` – PlantUML diagram for the high-level value chain.
